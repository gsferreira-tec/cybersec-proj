# Executing the attack

- To run the attack, execute the following command from the shell of the attacker container:

` ./attack.sh <sender-email> <destination-email> <message> <sender-faker>`

- <sender-email> corresponds to the valid domain account to be checked at the receiver
- <destination-email> corresponds to the victim's account
- <message> corresponds to the text in the email
- <sender-faker> corresponds to the account you want to be displayed at the victim's inbox
